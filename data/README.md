# Datasets

Este proyecto utiliza datasets externos (NO incluidos en GitHub por tamaño).

## Dataset manuscrito (colaborativo)
- Ubicación: Google Drive (carpeta compartida de clase)
- Contenido: imágenes de letras y números manuscritos por compañeros.

### Cómo usarlo
1. Descarga la carpeta del Drive (ZIP o sincronización).
2. Colócala en:

`data/raw/handwritten_colleagues/`

Estructura esperada (ejemplo):
- `data/raw/handwritten_colleagues/mayusculas/`
- `data/raw/handwritten_colleagues/minusculas/`
- `data/raw/handwritten_colleagues/numeros/`

> Nota: `data/raw/` está ignorado por git (no se sube).

## Muestras pequeñas
Para pruebas rápidas, se incluyen pocas imágenes en:
`data/samples/`
